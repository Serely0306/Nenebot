# 骗子酒馆 (PVP)

这是将 `astrbot_plugin_liars_bar` 的 PVP移植到 ZeroBot 插件框架下的 Go 版本。

快速使用：在群聊输入命令

- `/骗子酒馆` : 创建新游戏（群内）
- `/加入` : 加入当前等待中的游戏
- `/开始` : 发起者开始游戏（至少 2 人）
- `/出牌 <编号...>` : 出牌，例如 `/出牌 1 2`
- `/质疑` : 质疑上家
- `/等待` : 空手时选择等待
- `/状态` : 查看游戏状态
- `/我的手牌` : 私信查看你的手牌
- `/结束游戏` : 强制结束当前游戏

说明：
- 私信功能依赖 ZeroBot 的私信接口，请确保机器人能向玩家发送私信。
